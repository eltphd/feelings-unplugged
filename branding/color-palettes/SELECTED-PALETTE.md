# Selected Color Palette: Terracotta Forest

**Blending Option 1 (Terracotta + Sage) + Option 4 (Forest + Amber)**

## Color Swatches

```
Primary (Terracotta):     #E07A5F    [Warm, inviting, healing]
Secondary (Forest Green): #3A5A40    [Grounded, stable, growth]
Accent (Amber):           #D4A574    [Warm glow, hope, firefly light]
Supporting (Sage):        #81B29A    [Calm, balance, breath]

Background (Warm Cream):  #F4F1DE    [Soft, warm, approachable]
Surface (Off-White):      #FDFBF7    [Clean but not stark]
Text (Deep Charcoal):     #2D3142    [Readable, grounded]
Text Secondary (Gray):    #5A5A5A    [Softer for body text]

Border (Subtle):          #E8E3D8    [Warm neutral border]
Shadow (Soft):            rgba(45, 49, 66, 0.1)  [Gentle depth]
```

## Usage Guide

### Primary Terracotta (#E07A5F)
- Main CTAs ("Buy Now", "Join Waitlist")
- Headings (h2)
- Interactive elements on hover
- **Emotion:** Warm, inviting, healing, action

### Secondary Forest Green (#3A5A40)
- Subheadings (h3)
- Secondary buttons
- Navigation elements
- **Emotion:** Grounded, stable, trustworthy

### Accent Amber (#D4A574)
- Tags ("Coming Soon", page counts)
- Icons and sparkles (✨)
- Subtle glow effects
- **Emotion:** Hopeful, glowing, gentle light

### Supporting Sage (#81B29A)
- Borders on example boxes
- Calm/grounding sections
- Success states
- **Emotion:** Breathing space, calm, balance

## Accessibility

✅ **All combinations pass WCAG AA contrast:**
- Terracotta (#E07A5F) on Cream (#F4F1DE): 3.8:1 (AA Large Text)
- Forest (#3A5A40) on Cream (#F4F1DE): 7.2:1 (AAA)
- Text (#2D3142) on Cream (#F4F1DE): 11.5:1 (AAA)
- Terracotta on White: 4.2:1 (AA)
