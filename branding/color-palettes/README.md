## How to Use This Folder

1. **Save color palette screenshots** from sites/brands you love
2. **Export palette swatches** from Coolors, Adobe Color, etc.
3. **Name files descriptively:** `terracotta-sage-combo.png`, `headspace-palette.jpg`

## Tools for Exploring Palettes

- **Coolors.co** - Generate random palettes, adjust to taste
- **Adobe Color** - Extract colors from images
- **Huemint** - AI-generated color schemes
- **Pinterest** - Search "mental health branding colors"

## Guidelines

**For mental health/wellness:**
- Warm neutrals > stark whites
- Muted/soft colors > neon/bright
- Good contrast for accessibility (WCAG AA minimum)
- Colors that feel like a warm hug, not a clinical space
