# Reference Images

## Folder Structure

```
reference-images/
├── color-inspiration/       # Sites/brands with colors you love
├── layout-inspiration/      # Page layouts that feel right
├── mental-health-brands/    # Mental health sites doing it well
├── NOT-this/               # Examples of what to AVOID
└── misc/                   # Other inspiration
```

## How to Use

1. **Browse the web** - Save screenshots of sites that feel right
2. **Pinterest boards** - Search "mental health website design", "wellness branding"
3. **Save with notes** - Rename files to explain what you like

Example:
- `calm-app-homepage.png` → "Love the soft blue + coral combo"
- `headspace-cards.jpg` → "Card hover effects are perfect"
- `betterhelp-NO.png` → "Too clinical, too corporate"

## Suggested Searches

- "Mental health app design"
- "Wellness website inspiration"
- "Therapy practice branding"
- "Teen mental health resources design"
- "Indie book cover design" (for warmth + character)
- "Modern apothecary branding" (for healing + natural vibes)

## Questions to Ask When Saving

- What emotion does this evoke?
- Is it warm or cold?
- Is it approachable or intimidating?
- Would a teen in crisis feel safe here?
- Does it feel authentic or performative?
